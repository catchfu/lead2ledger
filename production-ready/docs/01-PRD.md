# Product Requirements Document

## Product
CRM + SAP for Small Businesses

## Problem
SMBs need one system for pipeline, customer operations, and finance/inventory execution without enterprise complexity.

## Objectives
- Unified CRM (leads, accounts, opportunities, activities).
- Bidirectional SAP integration for customers, products, quotes/orders, invoices, payments.
- Production-grade auditability, security, and deployment.

## Users
- Sales reps
- Sales managers
- Operations/admin
- Finance/accounting

## MVP Scope
- Lead-to-cash CRM pipeline
- Customer 360 view
- SAP customer/product sync
- Quote/order/invoice sync with status tracking
- Role-based access control and audit log

## Non-Goals (MVP)
- Custom ERP module development
- Complex CPQ engine
- AI forecasting beyond basic reports

## Success Metrics
- >95% sync success rate
- <5 min data propagation SLA
- <1% duplicate customer records
- P95 API latency < 300ms
